    프로젝트 개요

    자신이 자신있어하는 적성과 그와 관련된 직업군을 선택하면 그에 맞는 가장 적절한 직업들을 찾아
    직업의 연봉, 전망, 고용평등률 등의 정보를 알려주고 이 정보들을 이용해 추가적인 분류를 할 수 있는
    직업정보 제공 웹페이지를 구축하려 합니다.

    사용자와 과업 분석
    1. 페르소나

      예상되는 사용자(페르소나)는 대체적으로

      • 진로탐색을 하는 중, 고등학생.
      • 고등학교를 졸업했거나 대학을 진학중인 취업준비생.
      • 기존의 직업을 그만두고 새 길을 찾는 구직자.

      이 될 것 이라 예상했습니다.

    활용하고자 하는 공공 개방 데이터 소개 및 테스트

    커리어넷의 직업정보 데이터를 OpenAPI로 제공받았습니다.
    커리어넷 직업분류 혹은 적성유형을 입력해 그에 맞는 직업 데이터를 xml 형식으로 전송해줍니다.
    적성유형을 통해 얻는 정보에 직업분류도 포함되어 있어 적성유형을 입력받은 후
    직업분류로 2차 분류를 할 예정입니다.
    Link : 커리어넷 OpenAPI
    Link : 테스트 웹페이지

    대화형 프로토타입

    UI는 인터넷 강의에서 배운 jquery UI를 사용하였습니다.
    사용자 입력에 따라 자료들을 파싱하고 파싱한 자료를 사용자 입력으로 2차적으로 가공해
    더 정확한 정보를 얻어내고자 했습니다.
    결과 자료를 정렬 방식에 따라 다르게 보여주는 것도 추가하였습니다.
    Link : Visual Prototype

결론 및 향후 개선방향

특성선택, 직종선택, 결과창을 나눠 보여주는 기능을 아직 넣지 못하고
원하는 항목 체크시 아코디언을 닫아버리면 다시 열지 않는이상 체크 여부를 확인하는데 번거로움이 있는 부분과
입력값과 결과값의 연계성을 좀 더 시각적으로 보여주기 위한 방법을 강구해보려 합니다.
데이터 시각화에 아직 공부가 모자라 시각적 프로토타입에서 제안한 것들 중 추가하지 못한 부분이 있습니다.
추후 추가제출 때 추가적으로 넣고싶은 기능(텍스트입력 검색을 통한 직업 찾기 등)과 사용성 평가를 통해 컴플레인을 받아
완성도를 높일 예정입니다.
